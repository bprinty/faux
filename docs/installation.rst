============
Installation
============


Through pip
-----------


.. code-block:: bash

    $ pip install gems


Via GitHub
----------


.. code-block:: bash

    $ git clone http://github.com/bprinty/smock.git
    $ cd smock
    $ python setup.py install

------------------

File an issue in the `GitHub issue tracker <https://github.com/bprinty/smock/issues>`_.
